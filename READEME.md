# Vision Data Pipeline

An automated image data engineering pipeline for computer vision datasets.

## Features

- Dataset loading
- Dataset profiling
- Data quality analysis
- Image standardization
- ResNet50 feature extraction
- Metadata generation

## Architecture

Dataset

↓

Analysis

↓

Quality Check

↓

Standardization

↓

Feature Extraction

↓

Metadata
